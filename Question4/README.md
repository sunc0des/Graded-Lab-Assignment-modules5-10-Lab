Question 4

This folder contains:
- emailcleaner.sh
- emails.txt
- valid.txt
- invalid.txt
- Screenshots

  
- Explanations for each command:

- Command:
nano emails.txt

Explanation:
This command creates the input file containing email addresses.

Command:
nano emailcleaner.sh

Explanation:
This command creates the shell script that separates valid and invalid emails.

Command:
chmod +x emailcleaner.sh

Explanation:
This command gives execute permission to the script.

Command:
./emailcleaner.sh

Explanation:
This command runs the script and generates valid.txt and invalid.txt files.

Command:
cat valid.txt

Explanation:
This command displays all valid email addresses without duplicates.

Command:
cat invalid.txt

Explanation:
This command displays all invalid email addresses.
